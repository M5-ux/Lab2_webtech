 
                Fonction pour API 
 
 -const express = require('express');: Cette ligne importe le module Express.js,
 qui est un framework pour créer des applications web en Node.js.
 Elle permet de créer un objet express que nous utiliserons pour configurer notre application.

 -const router = express.Router();: Nous créons ici un routeur Express en utilisant express.Router().
  Un routeur est utilisé pour organiser nos routes de manière modulaire.

  app.use('/', router);


 Cette ligne indique à l'application Express d'utiliserle routeur router que nous 
 avons défini précédemment pour toutes les routes qui commencent par 
 Cela permet de monter les routes du routeur sur le chemin racine de l'application.


app.listen(port, () => { ... });: Cette ligne démarre le serveur Express en écoutant
  les demandes entrantes sur le port spécifié (port). Lorsque le serveur démarre avec succès, 
  la fonction de rappel est exécutée, affichant un message indiquant que le serveur est 
  en cours d'exécution.
